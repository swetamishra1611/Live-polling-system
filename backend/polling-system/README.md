# Polling System

A basic Express.js app for a polling system.

## Available Scripts

- `npm start` — Start the server

## Health Check

- GET `/health` — Returns `{ status: 'OK' }`

## Getting Started

1. Install dependencies:
   ```sh
   npm install
   ```
2. Start the server:
   ```sh
   npm start
   ```

The server will run on port 3000 by default.
